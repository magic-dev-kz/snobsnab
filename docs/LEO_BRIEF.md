## Задачи для Лео

### 1. 🎨 Визуальная доработка лендинга
Текущий лендинг — функциональный, но без фотографий и визуальных акцентов.
- Добавить качественные фотографии стройматериалов в секции каталога
- Проработать hero-секцию: фон-фото или иллюстрация строительной тематики
- Проверить мобильную адаптацию (Tailwind responsive)
- Анимации при скролле (опционально, не перегружать)

### 2. 📦 Страницы категорий каталога
Сейчас каталог — это 6 карточек на главной с иконками. Нужно:
- Создать отдельные страницы для каждой категории: `/catalog/[category]`
- Структура страницы категории:
  - Заголовок + описание категории
  - Список товаров (карточки): фото, название, краткое описание, кнопка "Запросить цену"
  - Фильтры (опционально)
- **Контент товаров** — пока можно демо-данные, потом Илья наполнит реальными

### Категории:
1. **Кровельные материалы** — металлочерепица, профнастил, гибкая черепица, мембраны
2. **Утеплители** — минвата, экструзия, пенопласт, PIR-плиты
3. **Сухие смеси** — штукатурки, клеи, наливные полы, затирки
4. **Фасадные системы** — сайдинг, фиброцемент, вентфасады, декор
5. **Гидроизоляция** — битумные, полимерные, обмазочные, рулонные
6. **Крепёж и метизы** — саморезы, анкеры, дюбели, профили

## Файловая структура
```
src/
  pages/
    index.astro          ← главная (лендинг)
    blog/
      index.astro        ← список статей
      [slug].astro       ← шаблон статьи
    api/
      contact.ts         ← форма → Telegram
  layouts/
    BaseLayout.astro     ← общий layout
  content/
    config.ts            ← Content Collections
    blog/                ← статьи .md
  styles/
    global.css           ← Tailwind + кастомные стили
public/
  logo.svg, favicon.svg
  blog/                  ← обложки статей
```

## Как работать
1. Клонировать: `git clone https://github.com/magic-dev-kz/snobsnab.git`
2. `npm install && npm run dev` — локальный сервер
3. Push в main → Vercel деплоит автоматически
